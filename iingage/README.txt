iingage - A simple ii irc bot - makes logs and more!

Requirements:
	Bash
	ii installed (http://tools.suckless.org/ii/)
	
Using iingage:
	~Commands for iingage~
	<bot nickname>, gen html 	: Generate an HTML page of the current logs
	<bot nickname>, up <server>	: Ping a server and reports if online or offline
	<bot nickname>, eat <nick>	: Eat whoever you want it too :D
	<bot nickname>, die 		: Stop iingage
	<bot nickname>, server load	: Report the current server load.
